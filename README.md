# Brick-block-Game

A falling block puzzle game built using **Python and Pygame**.  
The player places different types of blocks as they fall and tries to complete horizontal lines. When a full line of blocks is created, it gets cleared and the player earns points.

## 🎮 Game Features

- Built using Pygame
- Falling block mechanics
- Multiple block shapes
- Grid-based gameplay system
- Line detection and clearing system
- Score tracking
- Next block preview
- Play and reset controls
- Custom game interface with separate information panel and gameplay area

## 🛠️ Technologies Used

- Python
- Pygame
